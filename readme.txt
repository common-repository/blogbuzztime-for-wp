BlogBuzzTime displays current readers on your blog, in real time, to other readers.

Affiche les articles en cours de lecture sur votre blog, en temps réel.

== Description ==

BlogBuzzTime displays a widget in your sidebar with current readers on your blog.

This is REAL TIME: each time an user leave, or user comes on an article, all your readers will see updated counter.

This plugin provides 2 widgets:

- 1 showing how many users are online on your blog

- 1 showing current readers

BlogBuzzTime vous permet d'afficher les articles en cours de lecture sur votre blog, en temps réel, ainsi que le nombre total d'internautes en ligne.


== Installation ==

Install this plugin as many others.

1. Upload all the zip files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget using the widget menu.
